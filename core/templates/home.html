Django + Docker + CI/CD + Render Deployment Steps with Commands

Step 1

Project Setup (Django)

We created a Django project and app structure.

django-admin startproject myproject
cd myproject
python manage.py startapp core
python manage.py runserver
Step 2

Docker Setup

We containerized the Django application using Docker.

docker build -t django-app .
docker run -p 8000:8000 django-app

Dockerfile used:

FROM python:3.11
WORKDIR /app
COPY . /app
RUN pip install -r requirements.txt
CMD ["python", "manage.py", "runserver", "0.0.0.0:8000"]
Step 3

Git Version Control

We tracked changes using Git and pushed to GitHub.

git init
git add .
git commit -m "initial commit"
git branch -M main
git remote add origin https://github.com/user/repo.git
git push -u origin main
Step 4

CI/CD Pipeline

Automated build and deployment workflow.

name: Django CI

on: [push]

jobs:
  build:
    runs-on: ubuntu-latest

    steps:
      - uses: actions/checkout@v3
      - name: Install dependencies
        run: pip install -r requirements.txt
      - name: Run tests
        run: python manage.py test
Step 5

Deployment (Render)

We deployed the project to Render cloud platform.

Build Command: pip install -r requirements.txt
Start Command: python manage.py runserver 0.0.0.0:8000
Step 6

Final Testing

We tested the live deployment for routing, templates, and APIs.

curl http://your-app.onrender.com
# or open in browser
